# Hangman game

## Play

```sh
$ python hangman.py
```

or 

```sh
$ python hangman.py [language]
```

#### Example

```sh
$ python hangman.py pt-br
```

## Node.js 

```sh
$ node hangman.js
```

or 

```sh
$ node hangman.js [language]
```

#### Example

```sh
$ node hangman.js pt-br
```
